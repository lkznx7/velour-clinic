# Velour Clinic - Website

Website institucional para clínica de estética premium oferecendo tratamentos faciais e corporais.

## Problema que Resolve

Mulheres e homens que buscam tratamentos estéticos de qualidade precisam conhecer os serviços disponíveis, entender os benefícios de cada procedimento e agendar consultas. O site serve como funil de conversão para captação de novos clientes e agendamentos.

## Tipo

Landing Page Institucional

## Stack

- **Framework:** Next.js 15 (App Router)
- **Styling:** Tailwind CSS v4
- **Language:** TypeScript
- **部署:** Vercel

## Status

Em desenvolvimento

```bash
npm install
npm run dev
```
